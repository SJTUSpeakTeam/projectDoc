**上传图片/文件**

````

Request: POST /file/upload
body: {
	RawBlobData // 将 raw blob 数据直接上传，不要打包 JSON
}

Response:
data(successful): {
    "status": Integer // -1 为失败， 0 为成功
    "msg": String // 例子："操作成功 : )",
    "data": {
    	"link" : String // 即 图片/文件 的获取链接
    }
}
````

## 一、用户管理

**用户状态须知：**

````
用户状态：user_type or userType
内容：Integer
分类：
0: 被 ban 用户
1: 管理员用户
2: 正常用户
````

**登录**

````
Auth: None
Request: POST /login
Body: {
    "mailAddr": String,
    "password": String
}

Response:
data(successful): {
    "status": Integer, // -1 为失败， 0 为成功
    "msg": String, // 例子："登录成功 :)"
    "data": {
        "avatar": String,  // 这是个 url
        "createTime": Datetime ,
        "gender": Boolean ,
        "mailAddr": String,
        "nickname": String,
        "phoneNum": String,
        "userId": Integer,
        "token": String,
        "email": String,
        "userType": Integer // 用来判断跳转去用户还是管理员界面
    }
}
````
