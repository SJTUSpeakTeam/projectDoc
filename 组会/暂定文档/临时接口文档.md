# 二、帖子管理
* #### 问题发布 ×
```matlab
Request: POST /postQuestion
Body: {
    *header: string
    *content: string
    *userId: integer
    *partitionId: integer
    tag: [string, ...]
}

Response:
data(successful): {
    questionId: integer
}
```

* #### 问题获取 ×
```matlab
Request: GET /getQuestion
Parameters: questionId(integer)

Response:
data(successful): {
    questionId: integer
    userId: integer
    partitionId: integer
    header: string
    content: string
    createTime: date
    likeNum: integer
    favorNum: integer
    visitNum: integer
    status: short
}
```

* #### 问题更改 ×(只允许更改对应tag和partition)
```matlab
Request: POST /updateQuestion
Body: {
    *questionId: integer,
    *partitionId: integer
    tag: [string, ...]
}

Response:
data(successful): {
    questionId: integer
}
```

* #### 问题删除 ×
```matlab
Request: DELETE /postQuestion
Body: {
    *questionId: integer
}

Response:
data(successful): {}
```

* #### 问题点赞 ×
```matlab
Request: GET /likeQuestion
Parameters: userId(integer),questionId(integer),like(boolean)

```

* #### 问题收藏 ×
```matlab
Request: GET /collectQuestion
Parameters: userId(integer),questionId(integer),collect(boolean)

```

* #### 增加问题访问次数 ×
```matlab
Request: GET /browseQuestion
Parameters: userId(integer),questionId(integer)
```
